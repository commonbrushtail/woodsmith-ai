# WoodSmith Website — Checklist สำหรับกรอกข้อมูล

**เว็บไซต์**: https://woodsmith.vercel.app
**Admin Panel**: https://woodsmith.vercel.app/admin

---

## ขั้นตอนที่ 1: ตั้งค่าเว็บไซต์ (Site Settings)

**เมนู**: จัดการเนื้อหา → ตั้งค่าเว็บไซต์ (Site Settings)
**URL**: `/admin/site-settings`

### ข้อมูลบริษัท
- [ ] ชื่อบริษัท
- [ ] ที่อยู่บริษัท
- [ ] เบอร์โทรศัพท์

### โซเชียลมีเดีย
- [ ] LINE ID (เช่น @vanachai.woodsmith)
- [ ] LINE URL (ลิงก์เพิ่มเพื่อน)
- [ ] Facebook URL
- [ ] Instagram URL
- [ ] TikTok URL

### สถิติหน้าแรก
- [ ] จำนวนสาขา (เช่น 97)
- [ ] จำนวนสินค้า (เช่น 20+)
- [ ] จำนวนลูกค้า (เช่น 10K+)

### ข้อความส่วนท้าย
- [ ] ข้อความลิขสิทธิ์ (เช่น © 2019 @vanachai.woodsmith. All rights reserved.)

### รูปแบนเนอร์หน้าเว็บ (แนะนำ 1920x400px)
- [ ] แบนเนอร์หน้าเกี่ยวกับเรา (About)
- [ ] แบนเนอร์หน้าบทความ (Blog)
- [ ] แบนเนอร์หน้าคู่มือ (Manual)
- [ ] แบนเนอร์หน้าไฮไลท์ (Highlight)
- [ ] แบนเนอร์หน้าคำถามที่พบบ่อย (FAQ)

## ขั้นตอนที่ 2: เกี่ยวกับเรา (About Us)

**เมนู**: จัดการเนื้อหา → เกี่ยวกับเรา (About Us)
**URL**: `/admin/about-us`

- [ ] เขียนเนื้อหาเกี่ยวกับบริษัท (รองรับจัดรูปแบบตัวอักษร, แทรกรูปภาพ)

---

## ขั้นตอนที่ 3: ช่องทางสาขา (Branches)

**เมนู**: จัดการเนื้อหา → ช่องทางสาขา (Branch)
**URL**: `/admin/branch`

สำหรับแต่ละสาขา:
- [ ] ชื่อสาขา
- [ ] ที่อยู่
- [ ] เบอร์โทรศัพท์
- [ ] พิกัด GPS (latitude / longitude)
- [ ] ลิงก์ Google Maps
- [ ] รูปภาพสาขา
- [ ] เวลาเปิด-ปิด
- [ ] รายละเอียดเพิ่มเติม

**จำนวนสาขาที่ต้องกรอก**: _____ สาขา

---

## ขั้นตอนที่ 4: แบนเนอร์หน้าแรก (Banner)

**เมนู**: จัดการเนื้อหา → แบนเนอร์ (Banner)
**URL**: `/admin/banner`

สำหรับแต่ละแบนเนอร์:
- [ ] รูปภาพแบนเนอร์ (แนะนำ 2560x1440px)
- [ ] ลิงก์ URL ที่ต้องการให้กดไป (ถ้ามี)

**จำนวนแบนเนอร์ที่ต้องการ**: _____ รูป

---

## ขั้นตอนที่ 5: แกลลอรี่ (Gallery)

**เมนู**: จัดการเนื้อหา → แกลลอรี่ (Gallery)
**URL**: `/admin/gallery`

- [ ] รูปภาพแกลลอรี่หน้าแรก (Homepage Gallery) — หลายรูป, ลากจัดลำดับได้
- [ ] รูปภาพแกลลอรี่หน้าเกี่ยวกับเรา (About Gallery) — หลายรูป, ลากจัดลำดับได้

---

## ขั้นตอนที่ 6: คำถามที่พบบ่อย (FAQ)

**เมนู**: จัดการเนื้อหา → คำถามที่พบบ่อย (FAQs)
**URL**: `/admin/faq`

สำหรับแต่ละคำถาม:
- [ ] คำถาม
- [ ] คำตอบ (รองรับจัดรูปแบบตัวอักษร)
- [ ] หมวดหมู่ (ถ้ามี)

**จำนวนคำถามที่ต้องกรอก**: _____ ข้อ

---

## ขั้นตอนที่ 7: ตรวจสอบสินค้า (Products)

**เมนู**: จัดการเนื้อหา → สินค้า (Product)
**URL**: `/admin/products`

> มีสินค้า 38 รายการที่โหลดไว้แล้ว — กรุณาตรวจสอบความถูกต้อง

สำหรับแต่ละสินค้า ตรวจสอบ:
- [ ] ชื่อสินค้าถูกต้อง
- [ ] รหัสสินค้าถูกต้อง
- [ ] รูปภาพสินค้าครบถ้วน (เลือกรูปหลัก)
- [ ] คำอธิบายสินค้าถูกต้อง
- [ ] คุณลักษณะ / สเปค ถูกต้อง
- [ ] หมวดหมู่ถูกต้อง
- [ ] เปิด/ปิดเครื่องคิดพื้นที่ (Area Calculator) ตามต้องการ
- [ ] ตัวเลือกสินค้า (Variations) ถูกต้อง
- [ ] สถานะเผยแพร่ (Published) ถูกต้อง

### ประเภทสินค้า & หมวดหมู่
**URL**: `/admin/product-types` และ `/admin/categories`

- [ ] ตรวจสอบประเภทสินค้าครบถ้วน
- [ ] ตรวจสอบหมวดหมู่และหมวดหมู่ย่อยครบถ้วน

### ตัวเลือกสินค้า (Variations)
**URL**: `/admin/variations`

- [ ] ตรวจสอบกลุ่มตัวเลือก (เช่น ขนาด, สี, รุ่น) ครบถ้วน

---

## ขั้นตอนที่ 8: เนื้อหาเพิ่มเติม (เพิ่มทีหลังได้)

### บทความ (Blog)
**URL**: `/admin/blog`

ก่อนเขียนบทความ ให้สร้างหมวดหมู่บทความก่อน:
- [ ] สร้างหมวดหมู่บทความ (`/admin/blog-categories`)

สำหรับแต่ละบทความ:
- [ ] หัวข้อ
- [ ] เนื้อหา (รองรับจัดรูปแบบตัวอักษร, แทรกรูป)
- [ ] รูปปก (Cover Image)
- [ ] หมวดหมู่
### คู่มือการใช้สินค้า (Manual)
**URL**: `/admin/manual`

สำหรับแต่ละคู่มือ:
- [ ] หัวข้อ
- [ ] รูปปก (Cover Image)
- [ ] ไฟล์ PDF
- [ ] YouTube URL (ถ้ามี)

### วิดีโอไฮไลต์ (Video Highlight)
**URL**: `/admin/video-highlight`

สำหรับแต่ละวิดีโอ:
- [ ] หัวข้อ
- [ ] YouTube URL
- [ ] หมวดหมู่ (สัมภาษณ์ / สินค้า / อีเว้นท์ / วิธีใช้)
- [ ] ประเภท (แนะนำ / ยอดนิยม / ล่าสุด)
