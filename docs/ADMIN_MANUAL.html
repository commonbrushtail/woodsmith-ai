1 เข้าสู่ระบบ Admin

URL: woodsmith.vercel.app/admin

เปิดเว็บไซต์แล้วไปที่ /admin จะเจอหน้า Login ให้กรอกอีเมลและรหัสผ่านที่ได้รับ

หากลืมรหัสผ่าน สามารถกด "ลืมรหัสผ่าน?" ด้านล่างปุ่มเข้าสู่ระบบได้

2 หน้า Dashboard

หลังจาก Login สำเร็จจะเข้าสู่หน้านี้โดยอัตโนมัติ

หน้า Dashboard แสดงภาพรวมของเว็บไซต์ — จำนวนสินค้า, แบนเนอร์, บทความ, ใบเสนอราคา

หน้า Dashboard — ภาพรวมข้อมูลทั้งหมดในระบบ

ด้านซ้ายคือเมนูหลัก แบ่งเป็น 3 หมวด:

  • 📖 จัดการเนื้อหา — แบนเนอร์, ตั้งค่าเว็บ, บทความ, สินค้า ฯลฯ
  • 📄 จัดการใบเสนอราคา — ดูและจัดการคำขอใบเสนอราคาจากลูกค้า
  • 👥 จัดการผู้ใช้งาน — ดูรายชื่อผู้ใช้, จัดการบัญชี

3 ตั้งค่าเว็บไซต์ (Site Settings)

เมนู: จัดการเนื้อหา → ตั้งค่าเว็บไซต์ (Site Settings)

หน้านี้คือศูนย์กลางของการตั้งค่าเว็บไซต์ทั้งหมด — ข้อมูลบริษัท, โซเชียลมีเดีย, สถิติ, และรูปแบนเนอร์

ส่วนบน — ข้อมูลบริษัท, โซเชียลมีเดีย

สิ่งที่ต้องกรอก:

  • ชื่อบริษัท (Company Name)
  • ที่อยู่บริษัท (Company Address)
  • เบอร์โทรศัพท์ (Phone Number)
  • LINE ID เช่น @vanachai.woodsmith
  • LINE URL (ลิงก์เพิ่มเพื่อน)
  • Facebook URL
  • Instagram URL
  • TikTok URL
  • ข้อความลิขสิทธิ์ เช่น © 2019 @vanachai.woodsmith
  • จำนวนสาขา / จำนวนสินค้า / จำนวนลูกค้า (แสดงหน้าแรก)
ส่วนล่าง — รูปแบนเนอร์หน้าเว็บ (แนะนำ 1920x400px)

รูปแบนเนอร์หน้าเว็บ:

  • แบนเนอร์หน้าเกี่ยวกับเรา (About)
  • แบนเนอร์หน้าบทความ (Blog)
  • แบนเนอร์หน้าคู่มือ (Manual)
  • แบนเนอร์หน้าไฮไลท์ (Highlight)
  • แบนเนอร์หน้าคำถามที่พบบ่อย (FAQ)
รูปแบนเนอร์แนะนำขนาด 1920 x 400 พิกเซล เพื่อให้แสดงผลสวยงามทุกขนาดหน้าจอ
กรอกข้อมูลเสร็จแล้ว อย่าลืมกดปุ่ม "บันทึก" ด้านขวา

4 เกี่ยวกับเรา (About Us)

เมนู: จัดการเนื้อหา → เกี่ยวกับเรา (About Us)

เขียนเนื้อหาแนะนำบริษัท — รองรับจัดรูปแบบตัวอักษร (หนา, เอียง, หัวข้อ) และแทรกรูปภาพได้

หน้าเกี่ยวกับเรา — เขียนเนื้อหาแนะนำบริษัท
  • เขียนเนื้อหาเกี่ยวกับบริษัท (ประวัติ, วิสัยทัศน์, สิ่งที่ทำ)

5 ช่องทางสาขา (Branch)

เมนู: จัดการเนื้อหา → ช่องทางสาขา (Branch)

เพิ่มข้อมูลสาขาทั้งหมด กดปุ่ม "เพิ่ม" เพื่อสร้างสาขาใหม่

รายการสาขา — กดปุ่ม "เพิ่ม" เพื่อสร้างสาขาใหม่
ฟอร์มสร้างสาขาใหม่

สำหรับแต่ละสาขา:

  • ชื่อสาขา
  • ที่อยู่
  • เบอร์โทรศัพท์
  • พิกัด GPS (latitude / longitude)
  • ลิงก์ Google Maps
  • รูปภาพสาขา
พิกัด GPS หาได้จาก Google Maps — คลิกขวาบนแผนที่แล้วจะเห็นตัวเลข latitude, longitude

6 แบนเนอร์หน้าแรก (Banner)

เมนู: จัดการเนื้อหา → แบนเนอร์ (Banner)

แบนเนอร์สไลด์ที่แสดงอยู่ด้านบนสุดของหน้าแรก — สามารถใส่ลิงก์ URL ให้กดไปหน้าอื่นได้

รายการแบนเนอร์หน้าแรก

สำหรับแต่ละแบนเนอร์:

  • รูปภาพแบนเนอร์ (แนะนำ 2560 x 1440 px)
  • ลิงก์ URL ที่ต้องการให้กดไป (ถ้ามี)

7 แกลลอรี่ (Gallery)

เมนู: จัดการเนื้อหา → แกลลอรี่ (Gallery)

อัปโหลดรูปภาพสำหรับแกลลอรี่หน้าแรกและหน้าเกี่ยวกับเรา สามารถลากจัดลำดับรูปได้

จัดการรูปแกลลอรี่ — ลากเพื่อจัดลำดับ
  • อัปโหลดรูปแกลลอรี่หน้าแรก (Homepage Gallery)
  • อัปโหลดรูปแกลลอรี่หน้าเกี่ยวกับเรา (About Gallery)

8 คำถามที่พบบ่อย (FAQ)

เมนู: จัดการเนื้อหา → คำถามที่พบบ่อย (FAQs)

เพิ่มคำถาม-คำตอบที่ลูกค้าถามบ่อย กดปุ่ม "เพิ่ม" เพื่อสร้างคำถามใหม่

รายการคำถามที่พบบ่อย
ฟอร์มสร้างคำถาม-คำตอบใหม่

สำหรับแต่ละคำถาม:

  • คำถาม
  • คำตอบ (รองรับจัดรูปแบบตัวอักษร)
  • หมวดหมู่ (ถ้ามี)

9 สินค้า (Products) — ตรวจสอบข้อมูล

เมนู: จัดการเนื้อหา → สินค้า (Product)

มีสินค้า 33 รายการ ที่โหลดไว้แล้ว — กรุณาตรวจสอบความถูกต้องของแต่ละรายการ

รายการสินค้าทั้งหมด — คลิกที่สินค้าเพื่อแก้ไข
ฟอร์มสินค้า ส่วนบน — ชื่อ, รหัส, ประเภท, หมวดหมู่
ฟอร์มสินค้า ส่วนล่าง — รูปภาพ, คำอธิบาย, สเปค

สำหรับแต่ละสินค้า ตรวจสอบ:

  • ชื่อสินค้าถูกต้อง
  • รหัสสินค้าถูกต้อง
  • รูปภาพสินค้าครบถ้วน (เลือกรูปหลัก)
  • คำอธิบายสินค้าถูกต้อง
  • คุณลักษณะ / สเปค ถูกต้อง
  • หมวดหมู่ถูกต้อง
  • สถานะเผยแพร่ (Published)
สินค้าที่เป็น "ฉบับร่าง" จะไม่แสดงบนหน้าเว็บ — ต้องกดเผยแพร่ก่อน

10 ประเภทสินค้า & หมวดหมู่

เมนู: จัดการเนื้อหา → ประเภทสินค้า (Product Type) และ หมวดหมู่สินค้า (Category)

ตรวจสอบว่าประเภทสินค้าและหมวดหมู่ครบถ้วน — สินค้าต้องอยู่ในหมวดหมู่ที่ถูกต้อง

ประเภทสินค้า (Product Type) — ระดับบนสุด
หมวดหมู่สินค้า (Category) — ระดับย่อย
  • ตรวจสอบประเภทสินค้าครบถ้วน
  • ตรวจสอบหมวดหมู่และหมวดหมู่ย่อยครบถ้วน

11 ตัวเลือกสินค้า (Variations)

เมนู: จัดการเนื้อหา → ตัวเลือกสินค้า (Variations)

จัดการตัวเลือกสินค้า เช่น สี, ขนาด, รุ่น — เชื่อมโยงกับสินค้าแต่ละรายการ

รายการตัวเลือกสินค้า
  • ตรวจสอบกลุ่มตัวเลือก (เช่น ขนาด, สี, รุ่น) ครบถ้วน

12 บทความ (Blog)

เมนู: จัดการเนื้อหา → บทความ (Blog)

เขียนบทความเกี่ยวกับสินค้า, เทคนิคการใช้งาน, ข่าวสาร — เพิ่มทีหลังได้เรื่อยๆ

ก่อนเขียนบทความ ให้สร้าง หมวดหมู่บทความ ก่อน (เมนู: หมวดหมู่บทความ Blog Category)
สร้างหมวดหมู่บทความก่อน (Blog Category)
รายการบทความทั้งหมด
ฟอร์มเขียนบทความใหม่

สำหรับแต่ละบทความ:

  • หัวข้อบทความ
  • เนื้อหา (รองรับจัดรูปแบบตัวอักษร, แทรกรูป)
  • รูปปก (Cover Image)
  • เลือกหมวดหมู่

13 คู่มือการใช้สินค้า (Manual)

เมนู: จัดการเนื้อหา → คู่มือการใช้สินค้า (Manual)

อัปโหลดไฟล์ PDF คู่มือการใช้สินค้า พร้อมรูปปกและลิงก์ YouTube

รายการคู่มือทั้งหมด
ฟอร์มสร้างคู่มือใหม่

สำหรับแต่ละคู่มือ:

  • หัวข้อ
  • รูปปก (Cover Image)
  • ไฟล์ PDF
  • YouTube URL (ถ้ามี)

14 วิดีโอไฮไลต์ (Video Highlight)

เมนู: จัดการเนื้อหา → วิดีโอไฮไลต์ (Video Highlight)

เพิ่มวิดีโอจาก YouTube — สัมภาษณ์, แนะนำสินค้า, อีเว้นท์, วิธีใช้

รายการวิดีโอไฮไลต์
ฟอร์มเพิ่มวิดีโอใหม่

สำหรับแต่ละวิดีโอ:

  • หัวข้อ
  • YouTube URL
  • หมวดหมู่ (สัมภาษณ์ / สินค้า / อีเว้นท์ / วิธีใช้)
  • ประเภท (แนะนำ / ยอดนิยม / ล่าสุด)

15 ใบเสนอราคา (Quotations)

เมนู: จัดการใบเสนอราคา → จัดการใบเสนอราคา (Quotation)

ดูและจัดการคำขอใบเสนอราคาจากลูกค้า — ไม่ต้องกรอกข้อมูลเอง ลูกค้าจะส่งคำขอเข้ามา

รายการใบเสนอราคา — ดูและจัดการสถานะ

ดูรายชื่อผู้ใช้ทั้งหมด (Admin และลูกค้า)
