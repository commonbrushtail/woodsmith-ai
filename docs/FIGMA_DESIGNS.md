# Figma-to-Code Design Tracker

Track which Figma designs have been implemented as frontend code.

---

## How to Use

1. Paste the Figma URL for each design below
2. Mark status as designs are implemented
3. Use Figma MCP (`get_design_context`) to pull design into code

Status key: `done` = implemented, `in-progress` = being worked on, `pending` = not started
